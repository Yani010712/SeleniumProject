# SeleniumProject 
[Credit to the dev Avengers YouTube channel](https://www.youtube.com/watch?v=T3OvEcVvCRI)

The framework will be developed using:
- Maven
- Cucumber 
- Java
- Selenium Webdriver

### Set up & Running
- git clone https://github.com/Yani010712/SeleniumProject.git
- mvn clean verify
- mvn install
- mvn test
